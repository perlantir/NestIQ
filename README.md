# Quotient

Native iOS mortgage calculator for licensed loan officers. Five calculators
(Amortization, Income Qualification, Refinance Comparison, Total Cost
Analysis, HELOC vs Refinance), branded PDF export, AI-narrated summaries.
SwiftUI, iOS 18+, Swift 6.

> **Where the design and product specs live**
> - `DEVELOPMENT.md` вЂ” authoritative build spec.
> - `DECISIONS.md` вЂ” vendor / architecture / scope calls.
> - `design/design_handoff_quotient/` вЂ” read-only design handoff. README
>   wins over the CSS token files when they disagree.
> - `KICKOFF-SESSION-1.md` вЂ” session kickoff prompts.

## Build

You need: Xcode 16.x with the iOS 18 simulator runtime, Homebrew.

```bash
# Install required tooling once.
brew install xcodegen swiftlint xcbeautify
brew install muter-mutation-testing/formulae/muter   # nightly mutation tests only

# Regenerate the Xcode project after editing project.yml.
xcodegen generate

# Build the app for simulator.
xcodebuild -project Quotient.xcodeproj -scheme Quotient \
  -destination 'platform=iOS Simulator,name=iPhone 16' build
```

After cloning, the `.xcodeproj` is already checked in вЂ” you only need to run
`xcodegen generate` if you change `project.yml`.

> **Code signing.** Disabled for simulator builds (Apple Developer
> enrollment in flight). Set `DEVELOPMENT_TEAM` in `project.yml` once the
> team ID exists, then regenerate.

## Tests

The finance engine is the moat вЂ” it gets the bulk of the testing.

```bash
# Unit + property + golden fixture tests on the finance package.
cd Packages/QuotientFinance
swift test

# With coverage report.
swift test --enable-code-coverage
PROF=.build/arm64-apple-macosx/debug/codecov/default.profdata
BIN=.build/arm64-apple-macosx/debug/QuotientFinancePackageTests.xctest/Contents/MacOS/QuotientFinancePackageTests
xcrun llvm-cov report "$BIN" -instr-profile="$PROF" \
  -ignore-filename-regex='(Tests|.build)'
```
